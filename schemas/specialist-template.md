# Specialist Template Schema

Required sections, in order:

1. `# Specialist: <name>`
2. `## Base Identity`
3. `## Refinement Prompts`
4. `## Authority Boundaries`
5. `## Expected Output`

## Rules

- `## Base Identity` must define the reusable baseline role.
- `## Refinement Prompts` must contain at least one bullet or numbered example.
- `## Authority Boundaries` must state what the specialist does not own.
- `## Expected Output` must state what the spawning agent should expect back.
- Templates are starting points. They are not ready-to-spawn prompts without refinement.
